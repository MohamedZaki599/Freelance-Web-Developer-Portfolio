# دليل رفع المشاريع الخمسة بشكل منفصل على فيرسيل (Vercel Deployment Guide)

بما أن المشاريع الخمسة مجمعة في مستودع Git واحد (Monorepo)، يمكنك رفع كل مشروع بشكل مستقل ومنفصل تماماً على منصة **Vercel** للحصول على 5 روابط ديمو مختلفة لعملائك.

هناك طريقتان للقيام بذلك:

---

## 💻 الطريقة الأولى: من خلال موقع Vercel (الأسهل والأكثر استخداماً)

هذه الطريقة تربط مشاريعك بحساب GitHub الخاص بك ليتم تحديث الموقع تلقائياً على Vercel بمجرد دفع (Push) أي كود جديد.

### الخطوات:
1. قم برفع مستودعك المحلي على حساب GitHub الخاص بك باسم `Freelance-Web-Developer-Portfolio`.
2. اذهب إلى لوحة تحكم [Vercel Dashboard](https://vercel.com/) وسجل الدخول بحساب GitHub.
3. اضغط على زر **"Add New"** ثم اختر **"Project"**.
4. ستظهر لك قائمة بمستودعاتك على GitHub، اختر مستودع **`Freelance-Web-Developer-Portfolio`** واضغط **"Import"**.
5. **الخطوة الأهم (تحديد مجلد المشروع):**
   * في صفحة الإعدادات، ستجد خياراً يسمى **Root Directory**.
   * اضغط على زر **Edit** بجانبه، وحدد مجلد المشروع الذي تريد رفعه الآن (مثال: `01-dental-clinic`).
6. **إعدادات البناء (Build Settings):**
   * سيتعرف Vercel تلقائياً على أن المشروع مبني بـ **Next.js** ويقوم بضبط الإعدادات تلقائياً.
   * تأكد من أن **Framework Preset** مكتوب فيه `Next.js`.
7. اضغط على **"Deploy"**.
8. **لرفع باقي المشاريع:** كرر نفس الخطوات السابقة تماماً مع اختيار المجلد المناسب في كل مرة (مثل `02-egyptian-restaurant` وهكذا). ستحصل في النهاية على 5 مشاريع مختلفة في Vercel تشير كل منها لمجلد فرعي داخل نفس المستودع.

---

## 🛠️ الطريقة الثانية: باستخدام واجهة CLI من خلال الـ Terminal

إذا كنت تفضل رفع المشاريع مباشرة من جهازك دون الحاجة لربطها بـ GitHub أولاً:

### الخطوات:
1. افتح نافذة الأوامر (PowerShell أو CMD) وثبّت أداة Vercel CLI بشكل عام على جهازك بالمرور بالأمر التالي:
   ```bash
   npm install -g vercel
   ```
2. توجه إلى مجلد المشروع الذي ترغب في رفعه (مثال: عيادة الأسنان):
   ```bash
   cd 01-dental-clinic
   ```
3. قم بتشغيل أمر الرفع التفاعلي:
   ```bash
   vercel
   ```
4. سيطلب منك الـ CLI الإجابة على بعض الأسئلة لتهيئة المشروع:
   * **Set up and deploy?** اكتب `y` واضغط Enter.
   * **Which scope?** اختر حسابك الشخصي.
   * **Link to existing project?** اكتب `n` (لأنك تنشئ مشروعاً جديداً).
   * **What's your project's name?** اختر اسماً مناسباً للمشروع (مثل: `mohamed-zaki-dental`).
   * **In which directory is your code located?** اضغط Enter مباشرة لاختيار المجلد الحالي `./`.
   * **Want to modify settings?** اكتب `n` (ودع Vercel يتعرف على الإعدادات الافتراضية).
5. بعد انتهاء الرفع الأولي والتحقق من الرابط التجريبي، قم برفعه للإنتاج (Production) مباشرة بالأمر:
   ```bash
   vercel --prod
   ```
6. كرر نفس العملية لكل مجلد من المجلدات الخمسة.
