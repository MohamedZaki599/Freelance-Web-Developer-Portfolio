# 5 برومبت لبناء نماذج البورتفوليو

كل برومبت تحت جاهز تنسخه كامل وتلصقه في Claude أو Gemini أو Codex وقت ما تيجي تبني الديمو بتاعه. كل برومبت قائم بنفسه (Self-contained) — مش محتاج ترجع لبرومبت تاني.

**تلميح:** قبل ما تلصق أي برومبت، تقدر تغيّر اسم النشاط أو المدينة لو عايز الديمو يكون مخصص لعميل معين بدل النموذج العام.

---

## 1. 🦷 عيادة أسنان — السوق المحلي (أسيوط)

```
أنا عايز تبني لي صفحة كاملة لعيادة أسنان باستخدام Next.js 15 و TypeScript و Tailwind CSS (App Router).

## الفكرة
موقع تعريفي لعيادة أسنان في أسيوط، مصر، اسمها "عيادة النور لطب وتجميل الأسنان". الهدف: عميل يفتح الموقع من موبايله ويحس بالثقة والاحترافية من أول ثانية، ويقدر يحجز أو يتواصل بسهولة.

## الفئة المستهدفة
عائلات وأفراد في أسيوط وصعيد مصر، أعمار 25-55، بيدوروا على عيادة أسنان موثوقة وقريبة، بيهمهم النظافة والاحترافية والسعر الواضح.

## الهوية البصرية
- الألوان الأساسية: أزرق طبي هادئ (#2563EB) + أبيض (#FFFFFF) + رمادي فاتح للخلفيات (#F8FAFC)
- لون مساعد (Accent): فيروزي فاتح (#0EA5E9) للأزرار والتفاصيل
- الخط: Cairo أو Almarai من Google Fonts (واضح ومريح في القراءة)
- الإحساس العام: نظيف، طبي، مطمئن، احترافي بدون برود

## الأقسام المطلوبة (بالترتيب)
1. Navbar ثابت: شعار "عيادة النور" + روابط (الرئيسية - خدماتنا - عن العيادة - تواصل معنا) + زرار "احجز موعد" بارز
2. Hero Section: عنوان قوي "ابتسامتك أمانة عندنا" + جملة فرعية عن الخبرة والجودة + زرار CTA "احجز موعدك الآن" + صورة (placeholder)
3. قسم الخدمات: 6 كروت لخدمات (تنظيف الأسنان - حشوات - تقويم الأسنان - تبييض - زراعة الأسنان - علاج أسنان الأطفال) كل كارت بأيقونة وعنوان وجملة وصف قصيرة
4. قسم "ليه تختار عيادتنا": 3 نقاط (دكاترة متخصصين - أجهزة حديثة - أسعار واضحة بدون مفاجآت)
5. قسم عن العيادة: فقرة قصيرة عن الخبرة وسنوات العمل
6. قسم آراء العملاء: 3 تقييمات تجريبية بأسماء عربية وتقييم نجوم
7. قسم التواصل: رقم تليفون - رقم واتساب - العنوان - خريطة (placeholder) - مواعيد العمل
8. زرار واتساب عائم (Floating) في الجزء السفلي يمين الشاشة، شغال طول الموقع
9. Footer بسيط: حقوق النشر + روابط سوشيال ميديا

## المتطلبات التقنية
- Next.js 15 مع App Router، TypeScript، Tailwind CSS فقط (من غير مكتبات خارجية)
- اتجاه الصفحة RTL بالكامل (عربي من اليمين لليسار)
- Responsive 100% على الموبايل والكمبيوتر
- كل الكود في ملف app/page.tsx واحد
- استخدم بيانات نصية واقعية (مش Lorem Ipsum) بالعربي طبيعي ومناسب لعيادة أسنان مصرية
```

## 2. 🍽️ مطعم مصري — السوق المحلي (أسيوط)

```
أنا عايز تبني لي صفحة كاملة لمطعم مصري باستخدام Next.js 15 و TypeScript و Tailwind CSS (App Router).

## الفكرة
موقع تعريفي لمطعم أكل بيتي مصري في أسيوط اسمه "مطعم بيت الطعم". الهدف: زائر الموقع يحس بالدفا والشهية من أول نظرة، ويقدر يشوف المنيو ويطلب أو يحجز طاولة بسهولة.

## الفئة المستهدفة
عائلات وشباب في أسيوط، بيحبوا الأكل المصري البيتي، بيدوروا على مكان لمناسبات صغيرة أو غدا/عشا عادي، بيهمهم الجو والطعم الأصلي.

## الهوية البصرية
- الألوان الأساسية: برتقالي محروق/طوبي (#C2410C) + كريمي فاتح (#FEF3E2) + بني غامق للنصوص (#451A03)
- لون مساعد: أصفر ذهبي (#F59E0B) للتفاصيل والأزرار
- الخط: عنوان بخط Lalezar أو El Messiri (له طابع تراثي مصري)، النصوص العادية بخط Tajawal
- الإحساس العام: دافئ، بيتي، شهي، أصيل، مرحب

## الأقسام المطلوبة (بالترتيب)
1. Navbar: شعار "بيت الطعم" + روابط (الرئيسية - المنيو - من نحن - تواصل) + زرار "اطلب الآن"
2. Hero Section: عنوان "طعم البيت... في كل لقمة" + جملة فرعية عن الأكل البيتي الأصيل + زرار "شاهد المنيو" + صورة طبق شهي (placeholder)
3. قسم المنيو: أقسام (مقبلات - أطباق رئيسية - مشويات - حلويات) كل قسم فيه 3-4 أصناف بالاسم والسعر التقريبي وجملة وصف
4. قسم "قصتنا": فقرة قصيرة عن بداية المطعم وفلسفة الأكل البيتي
5. معرض صور (Gallery): شبكة من 6 صور placeholder للأطباق والمطعم
6. قسم آراء العملاء: 3 تقييمات تجريبية
7. قسم التواصل: العنوان - رقم واتساب للطلب - مواعيد العمل - خدمة التوصيل (نعم/لا)
8. زرار واتساب عائم لطلب سريع
9. Footer: حقوق النشر + سوشيال ميديا (انستجرام وفيسبوك)

## المتطلبات التقنية
- Next.js 15 مع App Router، TypeScript، Tailwind CSS فقط
- RTL بالكامل
- Responsive كامل على الموبايل
- كل الكود في app/page.tsx
- نصوص واقعية بالعربي المصري الطبيعي (أسماء أكلات مصرية حقيقية زي ملوخية، فراخ مشوية، كباب وكفتة، أم علي... إلخ)
```

## 3. ✨ عطور فاخرة — السوق الخليجي

```
أنا عايز تبني لي صفحة كاملة لبراند عطور فاخر باستخدام Next.js 15 و TypeScript و Tailwind CSS (App Router).

## الفكرة
موقع تعريفي لبراند عطور فاخر اسمه "دار العود" يستهدف السوق السعودي والخليجي. الهدف: إحساس فوري بالفخامة والتراث، وعرض المنتجات بشكل يخلي الزائر يحس إنه بيتسوق من براند عالمي.

## الفئة المستهدفة
عملاء في السعودية والإمارات والخليج، رجال ونساء 25-55، بيقدّروا العطور الفاخرة والعود الأصلي، عندهم قدرة شرائية عالية، بيهمهم الفخامة والهوية البصرية الراقية والأصالة.

## الهوية البصرية
- الألوان الأساسية: أسود (#0A0A0A) + ذهبي (#C9A227) + أبيض عظمي (#F5F0E8)
- لون مساعد: بني عميق (#3D2817) للتفاصيل
- الخط: العناوين بخط Aref Ruqaa أو Amiri (طابع كلاسيكي عربي راقي)، النصوص بخط Tajawal أو Cairo
- الإحساس العام: فخامة، تراث، رقي، حصرية (Exclusivity)

## محتوى ثنائي اللغة
الموقع لازم يكون بالعربي والإنجليزي معاً — زرار تبديل اللغة (AR/EN) في الـ Navbar.

## الأقسام المطلوبة (بالترتيب)
1. Navbar شفاف فوق صورة الـ Hero: شعار "دار العود / Dar Al Oud" + روابط (الرئيسية - مجموعاتنا - قصتنا - تواصل) + زرار تبديل اللغة
2. Hero Section: عنوان "عبق الأصالة... بلمسة عصرية" / "Heritage in Every Drop" + خلفية غامقة فخمة + زرار "اكتشف المجموعة"
3. قسم المجموعات المميزة: 3-4 كروت منتجات (اسم العطر، السعر بالريال السعودي، صورة placeholder، زرار "أضف للسلة")
4. قسم "قصة دار العود": فقرة عن التراث والحرفية وجودة المكونات (عود، مسك، عنبر)
5. قسم الأكثر مبيعاً: شبكة منتجات إضافية
6. قسم آراء العملاء: 3 تقييمات فاخرة الأسلوب
7. قسم التواصل: نموذج اشتراك بالنشرة البريدية + روابط انستجرام + واتساب للطلبات الخاصة
8. زرار واتساب عائم بتصميم ذهبي يناسب الهوية
9. Footer أسود: حقوق النشر + روابط سوشيال + سياسة الخصوصية (placeholder)

## المتطلبات التقنية
- Next.js 15 مع App Router، TypeScript، Tailwind CSS فقط
- دعم RTL/LTR حسب اللغة المختارة (استخدم useState بسيط لتبديل اللغة)
- Responsive كامل
- كل الكود في app/page.tsx
- نصوص واقعية فاخرة الأسلوب بالعربي والإنجليزي
```

## الأقسام المطلوبة (بالترتيب)

8. زرار واتساب عائم
9. Footer: حقوق النشر + روابط سوشيال + ساعات العمل

## المتطلبات التقنية
- Next.js 15 مع App Router، TypeScript، Tailwind CSS فقط
- RTL بالكامل
- Responsive كامل
- كل الكود في app/page.tsx
- نصوص واقعية رسمية بالعربي تناسب مكتب محاماة محترم

## الهوية البصرية

- لون مساعد: برتقالي مشمس (#F97316) للأزرار والـ CTA

9. زرار واتساب عائم
10. Footer: حقوق النشر + روابط سوشيال ميديا

## المتطلبات التقنية
- Next.js 15 مع App Router، TypeScript، Tailwind CSS فقط
